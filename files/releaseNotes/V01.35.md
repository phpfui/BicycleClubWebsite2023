 ## Version 1.35 BETA (2026-04-07)

 ### New Features
 - [Survey](/Survey) section added allowing crosstab analysis of survey data
 - File type is now displayed in the [File](/File/browse) section
 - [General Admission](/Admin/homePage) events can now be featured on the member home page
 - General Admission events can have separate membership extension dates per type of extension
 - [Ride Statistics](/Rides/Statistics/ride) now shows the total miles led per year
 - [Ride Leader](/Leaders/report) Report CSV download now shows leader address and mobile phone number
 - Only [PHP 8.5](https://www.php.net/releases/8.5/en.php) or higher is now supported
 - [Cron Jobs](/System/cron) now has a Run Scheduled Cron Jobs Now button for easy access from your phone
 - [Ride With GPS](/RWGPS/find) library routes now have a copy to clipboard button to make it easier to paste into ride listings

 ### Bug Fixes
 - Plus signs (+) are now stripped from members gmail.com address for better login experiences
 - General Admission registrants now get member pricing even if they have not signed in
