 ## Version 1.34 BETA (2026-02-11)

 ### New Features
 - Content Management has been improved to avoid leaving empty stories
 - You can now [Text All Ride Leaders](/Leaders/text) and [Text All Members](/Membership/textAll) if [Twilio SMS](https://www.twilio.com/) is enabled
 - Membership Audits now have a Delete All button to make for easier cleanup.
 - Join page now requires a Math [Captcha](https://en.wikipedia.org/wiki/CAPTCHA) as well as the Google Captcha to avoid spam joining
 - [General Admission](/GA/manage) events now allow better control of creating or extending memberships

 ### Bug Fixes
 - Uploading a new profile photo requires you to crop it
