 ## Version 1.36 BETA (2026-06-09)

 ### New Features
 - [Rides With](/Rides/with) now shows the count of rides with the other member
 - Added license plate number of list of signed up riders
 - General Admission Events now have a Last Registration Time
 - Rider Signup emails now show the ride name with a link to the sign up page
 - Elevation is now an optional field in Rides
 - Member phone contacts now allow direct cell call or texting
 - Rides can now use the Start Location for directions to start if different from RWGPS

 ### Bug Fixes
 - [Rides With](/Rides/with) now display the first year correctly
 - [Find Members](/Membership/find) now searches on gmail addresses correctly (removes . and + signs)
 - RWGPS syncs correctly for gmail addresses
 - The Tax on an invoice is set to zero if the order is completely paid by volunteer points
 - Uploading [Newsletters](/Newsletter/upload) now parses file names for dates better
