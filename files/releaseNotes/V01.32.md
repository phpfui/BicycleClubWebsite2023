 ## Version 1.32 BETA (2025-01-01)

 ### New Features
 - Event emails now have ~firstName~, ~lastName~, ~email~ substitution variables
 - Public Events now require the user to solve a CAPTCHA to avoid spam
 - Public Events have a Who's Comming page, but does not show last name, only initial
 - Better editing of Public Event registrants
 - [Membership Configuration](/Membership/Configure/configure) now has defaults for privacy settings

 ### Bug Fixes
 - Sign Up For This Ride button in a Ride Reminder email goes to the correct page if you are not signed in
 - Repeating a ride does not set unaffilated flag if person repeating the ride is not authorized for unaffilated rides
 - More error detection to prevent bad PayPal requests
 - PHP 8.5 support
